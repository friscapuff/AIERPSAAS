AIERP INVENTORY MODULE IMPLEMENTATION CHECKLIST

=== CORE ENTITIES ===
[x] Product Entity
    - id (UUID, primary key)
    - tenant_id (UUID, for multi-tenancy)
    - sku (string, unique per tenant)
    - name (string)
    - description (text)
    - category (string)
    - unit_of_measure (string)
    - unit_cost (decimal)
    - selling_price (decimal)
    - reorder_level (integer)
    - lead_time_days (integer)
    - is_active (boolean)
    - created_at (timestamp)
    - updated_at (timestamp)
    - created_by (UUID, user reference)
    - updated_by (UUID, user reference)

[x] Warehouse Entity
    - id (UUID, primary key)
    - tenant_id (UUID)
    - name (string)
    - location (string)
    - capacity (integer)
    - is_active (boolean)
    - created_at (timestamp)
    - updated_at (timestamp)

[x] Stock Entity
    - id (UUID, primary key)
    - tenant_id (UUID)
    - product_id (UUID, foreign key)
    - warehouse_id (UUID, foreign key)
    - quantity_on_hand (integer)
    - quantity_reserved (integer)
    - quantity_available (integer, calculated)
    - last_count_date (timestamp)
    - created_at (timestamp)
    - updated_at (timestamp)

[x] StockMovement Entity
    - id (UUID, primary key)
    - tenant_id (UUID)
    - product_id (UUID)
    - warehouse_id (UUID)
    - movement_type (enum: IN, OUT, ADJUSTMENT, TRANSFER)
    - quantity (integer)
    - reference_type (string, e.g., PO, SO, RMA)
    - reference_id (string)
    - notes (text)
    - created_by (UUID)
    - created_at (timestamp)
